O que é CSS?


Css, abreviação de Cascading Style Sheets, em português Folha de Estilo em Cascata, é uma linguagem de estilo. Sendo uma linguagem de estilo, ele é um decorador, enfeitador de uma página ou site.


Como funciona o CSS?


Funciona por meio de um código em que você pode fazer alterações rápidas de layout, como definição de cores, tamanhos de fonte, dentre outras coisas. Sua funcionalidade facilita a personalização do site, edição e também ajuda a diminuir a repetição na estrutura do código.
